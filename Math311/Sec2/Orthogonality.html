# Orthogonality for Vectors ## Introduction In a 2D space, such as a graph, two lines can be said to be "perpendicular" when they are at $90°$ angle. This is used to define distance, particularly between a line and a point. In larger degrees of space, or in more generalized vector spaces, we use **orthogonality** to serve a similar purpose; it is the analog of being perpendicular in $\mathbb{R}^2$ or $\mathbb{R}^3$. ## Definition for vectors First, let us define an operation called **scalar product**, which will let us define what it means to be orthogonal in $\mathbb{R}^n$. If the scalar product (sometimes called dot product in calculus) is 0, two vectors are orthogonal. ### Scalar Product > Definition: Scalar Product > Given two vectors $x = \left(\begin{array}{c}x_1 \\ x_2 \end{array}\right)$, $y = \left(\begin{array}{c}y_1 \\ y_2 \end{array}\right)$, the scalar product of $x$ and $y$ is: > $$x \cdot y = x^Ty = x_1y_1 + x_2y_2$$ For the scalar product, it is important to note that it is commutative, meaning you ca swap the order of the vectors: $$x^Ty = y^Tx$$ > Definition: Orthogonal > Two vectors are orthagonal ($x \perp y$) if $x^Ty = 0$. Now that we have these definitions, it will be useful to define length and distance. > Definition: Euclidean Length > Let $x = \left(\begin{array}{c}x_1 \\ x_2 \end{array}\right)$, the Euclidean length of $x$ is: > $$||x|| = \sqrt{x^Tx} = \sqrt{x_1^2 + x_2^2}$$ > Definition: Distance > Given $x, y \in \mathbb{R}^2$, the distance between $x$ and $y$ is: > $$||x - y|| = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2}$$ One more important thing to note is a property of the scalar product (given $x, y \in \mathbb{R}^2$): $$x^Ty = ||x|| * ||y|| * cos(\theta)$$ Where $\theta$ is the angle between the vectors. ## Uses The first use is to define a property analagous to perpendicularity. In $\mathbb{R}^2$ and $\mathbb{R}^3$, the two terms are equavalent, but orthagonality extends to all dimensions of vector. This has some interesting properties: 1. If $x$ and $y$ are orthagonal, then we have: $$x^Ty = 0$$ $$||x|| ||y||\cos(\theta) = 0$$ $$cos(\theta) = 0$$ $$\theta = 90° = \pi/2$$ 2. If $x,y$ are orthagonal: $$||x+y||^2 = ||x||^2 + ||y||^2$$ ~~Proof of 2 $$||x+y||^2 = (x^T + y^T)(x + y)$$ $$x^Tx + x^Ty + y^Tx + y^Ty$$ $$||x||^2 + 0 + 0 + ||y||^2$$ $$||x||^2 + ||y||^2$$ ~~ Note that this proof is valid for all dimensions. 3. Cauchy-Schwartz inequality $$|x^Ty| = ||x|| ||y|| \cos(\theta)$$ $$|x^Ty| <= ||x|| ||y||$$ ## Applications Because we have the $\cos(\theta)$ in the forumla for scalar product, we can get the angle between the two vectors as: $$\cos(\theta) = {x^Ty \over ||x|| ||y||}$$ ~~Example 1 Find the angle between $x = (3, 4)^T$ and $y = (-1, 7)^T$ $$\cos(\theta) = {x^Ty \over ||x|| ||y||}$$ $$\cos(\theta) = {3(-1) + 4(7) \over \sqrt{3^2 + 4^2} \sqrt{(-1)^2 + 7^2}}$$ $$\cos(\theta) = {25 \over \sqrt{25} \sqrt{50}}$$ $$\cos(\theta) = \sqrt{1 \over 2}$$ $$\theta = \arccos\left(\sqrt{1 \over 2}\right) = \cos^{-1}\left(\sqrt{{1 \over 2}}\right) = 45° $$ ~~ The other property of the scalar product is that it represents a projection of one vector onto the other.
