# Impulse Functions In physics, there will often be questions where there will be a large value over a very small time, such as the forces in a collision. To deal with these kinds of forces, physicists use the **Dirac delta function** ($\delta(t)$) (also called **unit impulse**) ~~an explanation of impulse If a force is a function of time, the the impulse is the integral with respect to time. $$I = \int\limits_a^b F(t)dt$$ If we represent the force as mass times the delta velocity: $$I = \int\limits_a^b m\frac{dv}{dt}dt = mv(b) - mv(a)$$ Impulse, therefore, is the change in momentum. ~~ Given a very large impulse occuring in a very small time, the $\delta(t)$ can be used to represent it. It is defined as the a function with area of 1 underneath and zero width: $$ \delta(t) = \begin{cases} 0 && t \ne 0 \\ \infty && t = 0 \end{cases} $$ This can be explained by taking a rectangle with area one and taking the limits as its width approaches infinity. The main use is called its "sifting property". By multiplying the delta function by any other function, it removes all values aside from the one where $\delta(T) \to T = 0$. $$\int\limits_{-\infty}^{\infty} \delta(t - c)f(t)dt = \frac{1}{\epsilon}*\epsilon f(c) = f(c)$$ Note that this uses a shift of $c$ to change where the value of $f(t)$ is taken. This is called "sifting" because it allows us to take the value of a function at only one point. This property makes it easy to compute the Laplace of $\delta(t)$: $$\mathcal{L}\{\delta(t - c)\} = \int\limits_0^{\infty} e^{-st}\delta(t - c)dt = e^{-sc}$$ This gives us Rule 26. Another important feature is that when taking the Laplace of a function multiplied by the delta function, we can treat $e^{-st}$ and the other function itself as $f(t)$, meaning we just take its value at $c$: $$\mathcal{L}\{f(t)\delta(t)\}=e^{-sc}f(c)$$ ~~Example 1 1. $\mathcal{L}\{\delta(t - \pi\} = e^{-\pi s}$ 2. $\mathcal{L}\{(t^3 - 2t)\delta(t - 3)\} = 21e^{-3s}$ 3. $\mathcal{L}^{-1}\left\{{s^2 \over s^2 + 1}\right\}$ This one can be split into $1 - {1\over s^2 + 1}$ by long division, which results in: $$\delta(t) - \sin(t)$$ ~~ ## Impulsive Force Definition If a force is very large over a very short time, then we can model such forces as: $$F(t) = I*\delta(t-c)$$ Notice that this gives the correct value for impulse, as integrating will result in $I$. ~~Example 2 - Physics Example A bat hits a baseball with mass 0.15 kg at time t = 3 sec, when the baseball has a speed of 40 m/s.
