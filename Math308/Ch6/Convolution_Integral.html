# The Convolution Integral The Laplace transform has been shown to be linear, meaning: $$\mathcal{L}\{f(t) + g(t)\} = F(t) + G(t)$$ But what happens when we multiply functions? $$\mathcal{L}\{f(t)g(t)\} \stackrel{?}{=} F(t)G(t)$$ Let us find out. (Spoiler: it is not, the solution is a bit more complex) !!disproof This is disproven in only one line. $$\mathcal{L}\{2\cdot 3\} = \frac{2}{s}\cdot \frac{3}{s} = \frac{6}{s^2} \ne \mathcal{L}\{6\} = \frac{6}{s}$$ !! ## Convolution Theorem The actual result is: $$F(s)G(S) = \mathcal{L}\{(f * g)(t)\}$$ where $$(f* g)(t) = \int_0^t f(t - \tau)g(\tau)d\tau = \int_0^t f(\tau)g(t - \tau)d\tau$$ Where $f * g$ is the **convolution** of $f$ and $g$. Notice that $(f * g)(0) = 0$. A quick formatting note: where convolution integrals are used in these notes, the $\cdot$ symbol will mean multiplication (not dot product), while $*$ will mean convolution. !!an example $$(f * g)(3) = \int_0^3 f(3 - \tau)g(\tau) d\tau$$ !! Essentially, this is saying that the inverse Laplace of two multiplied functions is the integral of their products where one has been replaced with $t - \tau$. $$\mathcal{L}^{-1}\{F(s)G(s)\} = \int_0^t f(\tau)g(t - \tau)d \tau$$ Where $\tau$ is a dummy variable. ## Properties of Convolution - $f * g = g * f$ - Commutatuive - $f * (g * h) = (f * g) * h$ - Associative - $f * (g + h) = f * g + f * h$ - Distributive (over addition) - $f * 0 = 0$
!!Example 1 $$(f * 1)(t) = \int_0^t f(t - \tau)d\tau = \int_0^t f(\tau_1)d\tau_1$$ Note that if you substitute $\tau_1 = t - \tau$ into the first integral, it will result in the second version (the two $\tau$s are not the same). So, if you use $f(t) = 1$, we can show that: $$1 * 1 = \int_0^t 1d\tau = t$$ $$1 * 1 * 1 = (1 * 1) * 1 = t * 1 = \int_0^t \tau d\tau = \frac{1}{2}t^2$$ If you continue this pattern, you get: $$1 * 1 = t$$ $$1 * 1 * 1 = \frac{1}{2}t^2$$ $$1 * 1 * 1 * 1 = \frac{1}{6}t^3$$ $$1 * 1 * 1 ... 1 = \frac{t^{n-1}}{(n-1)!}$$ Which corresponds to the Laplace: $$\frac{1}{s} \cdot \frac{1}{s} ... = \frac{1}{s^n}$$ $$\mathcal{L}^{-1}\left\{\frac{1}{s^n}\right\} = \frac{t^{n-1}}{(n-1)!}$$ Continuing this logic, take the integral of $1 * f(t)$: $$\mathcal{L}\{\int_0^t f(v)dv\} = \mathcal{L}\{1 * f(t)\} = \mathcal{L}\{1\} \cdot \mathcal{L}\{f(t)\} = {F(s) \over s}$$ !! !!More Examples Using the convolution theorem, solve: $$\mathcal{L}^{-1}\left\{{1 \over s^3 + s}\right\}$$ If we write this as the laplace of multiplied functions, we can use the convolution theorem: $$\mathcal{L}^{-1}\left\{{\frac{1}{s^2 + 1} \over s}\right\}$$ Here we have $\frac{F(s)}{s}$, so we can use that theorem. $$= \int_0^t \sin\tau d\tau = -\cos\tau$$ More examples: $$\mathcal{L}^{-1}\left\{{F(s) \over s^2}\right\} = \int_0^t f(\tau)(t - \tau)d\tau$$ $$\mathcal{L}^{-1}\left\{{F(s) \over s^2 + 1}\right\} = \int_0^t f(\tau)\sin(t - \tau)d\tau$$ !!
