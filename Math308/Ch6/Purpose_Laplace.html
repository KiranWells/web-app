# Purpose of Laplace Transform ## Solving Initial Value Problems The purpose of the Laplace transform in differential equations is that it changes the derivative to a normal function: ### Theorem If $f$ is continuous and of exponential order, and $f'$ is piecewise continuous on any interval. Then the Laplace transform of $f'$ exists for $s \gt a$, and: $$\mathcal{L}\{f'(t)\} = s\mathcal{L}\{f(t)\} - f(0)$$ This can be proved by using integration by parts: $$\mathcal{L}\{f'(t)\} = \int\limits_0^{\infty} e^{-st}f'(t)dt$$ $$= e^{-st}f(t)|_0^{\infty} - \int\limits_0^{\infty} -se^{-st}f(t)$$ $$= 0 - e^{0}f(0) - \mathcal{L}\{-sf(t)\}$$ $$= -f(0) + s\mathcal{L}\{f(t)\}$$ Remember, $s$ is considered to be a constant, and we are assuming the conditions above are true, namely $s > a$ for some $a$ that proves $f$ is of exponential order. We can also prove a similar formula for higher derivatives: $$\mathcal{L}\{f''(t)\} = s\mathcal{L}\{f'(t)\} - f'(0)$$ Which can be simplified to: $$\mathcal{L}\{f''(t)\} = s^2\mathcal{L}\{f(t)\} - sf(0) - f'(0)$$ So we have a continuing formula: $$ \begin{cases} f(t) && F(s) \\ f'(t) && sF(s) - f(0) \\ f''(t) && s(sF(s) - f(0)) - f'(0) \\ ... f^{(n)}(t) && s^nF(s) - \sum\limits_0^{n-1} s^{n - i - 1}f^{(i)}(0) \end{cases} $$ ## Inverse Laplace Transform We need to get back from the $s$ function to $t$, so we find an operation which does that: $$F(s) = \mathcal{L}\{f(t)\}$$ $$f(t) = \mathcal{L}^{-1}\{F(s\}$$ However, one caveat is that at jumps (discontinuities), the Laplace transform cannot determine the value *at* the jump.
